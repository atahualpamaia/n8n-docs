Transcript Embedding Generator (Daily)

This workflow generates vector embeddings for call transcripts stored in Supabase that do not yet have embeddings. It queries for unembedded records, batches them, sends them to OpenAI text-embedding-3-small for vectorization, and updates each record with its embedding. This powers semantic search across call transcripts in the Sales Intelligence dashboard.

Execution Flow
1
Fetch Unembedded Records — Queries Supabase for call_transcripts rows where the embedding column is null, up to 50 records per run.
2
Batch & Embed — Prepares transcript text into a batch payload and sends it to OpenAI Embeddings API (text-embedding-3-small, 1536 dimensions).
3
Update Records — Maps each embedding back to its source record and updates the Supabase call_transcripts table with the generated vector.
Technical Notes

Runs every 2 hours to catch new transcripts. Batch size capped at 50 to stay within OpenAI rate limits. Uses text-embedding-3-small (1536 dimensions) matching the pgvector HNSW index on call_transcripts.

Auto-Link Calendar Meetings to Deals

This workflow automatically links calendar meeting activities in Pipedrive to the most relevant open deal in the SALES 2026 pipeline (pipeline 33). When a meeting is created without a deal association, it extracts the prospect person ID from attendees, looks up their open deals, picks the best match (preferring same-rep ownership and most recently updated), and links the activity to that deal.

Execution Flow
1
Filter & Extract — Filters for meeting-type activities without a deal, then extracts the prospect person ID from the attendee list (excluding the organizer/user).
2
Deal Lookup — Queries the Pipedrive API for all deals associated with the prospect person, then selects the best open deal in pipeline 33 based on rep ownership and recency.
3
Link Activity — Updates the Pipedrive activity to associate it with the matched deal via the Pipedrive API.
Technical Notes

Only processes meetings without existing deal associations. Filters strictly to pipeline 33 (SALES 2026). Prefers deals owned by the same rep as the activity creator, falling back to most recently updated deal.
