ATIVO ⚡ Webhook POST (Fireflies notification on call completion)

(Fireflies) PID Call Analysis + Coach

Automatically analyzes every sales discovery call recorded via Fireflies.ai. Fetches the transcript, extracts Pain/Impact/Decision signals, scores each dimension with GPT-4o-mini, saves the result to Airtable, and emails the rep a PID report + personalized coaching message.

Fluxo de execução

1
Trigger & Fetch

Fireflies sends a webhook on call completion. The workflow fetches the full transcript via the Fireflies node.

2
Input Cleaning

Clean Input + Rep Map normalizes the transcript and maps the rep email to their name and nick (14 reps mapped).

3
PID Extraction

Two parallel GPT calls: PID Extraction extracts Pain, Impact, and Decision signals as structured JSON. Call Intelligence classifies call type, identifies the prospect, and extracts metadata.

4
Scoring (3 parallel GPT calls)

PID Score: Pain, PID Score: Impact, and PID Score: Decision each score their respective dimension on a 0–10 scale with criteria-level reasoning.

5
Calculate & Save

Calculate Scores merges the three scores into a normalized total. Airtable: Upsert Call saves the full record. Two emails are sent: PID Report (scores + evidence) and PID Coaching (personalized improvement tips).

ATIVO ⚡ Webhook POST (Ringostat call completion)

ATIVO ⚡ Webhook POST /send-user-data (Snov.io registration)

Inbound Leads Qualification

Processes every new Snov.io registration. Verifies email (polling loop), phone (Abstract API), person name and company name (GPT). Applies 5 ICP gates. Scores the lead. Routes qualified leads to Pipedrive (Person + Org + Deal) and sends to Clay for enrichment. All records persisted in Airtable.

Fluxo de execução

1
Trigger & Raw Record

Webhook receives registration payload. Airtable: Create Registration immediately creates a raw record.

2
Verification (6 parallel)

Email: Snov.io API with polling loop (up to 8×15s). Person name: GPT validation. Company name: GPT validation. Domain match: GPT check. Phone: Abstract API. Direct passthrough.

3
Record Assembly & ICP Gates (5 parallel)

Build Lead Record assembles all results, assigns team, calculates lead score and data score. Five gates: country, personal email, job position, main goals, data score (must be 100). All must pass.

4
Pipedrive Creation (2 parallel)

Search Person → update or create. Create Organization (always). Both merge before deal creation.

5
Deal + Enrichment

Create Deal with 16 custom fields. Save Deal ID to Airtable. Send to Clay (with retry). Check org duplicates.

ATIVO ⚡ Calendly webhook (Discovery Call booked)

Create/Update a deal from Discovery Call

When a prospect books a Discovery Call via Calendly, this workflow creates or updates their deal in Pipedrive. Verifies email and phone, searches for existing person, creates person/org if needed, creates or updates the deal, assigns the correct owner, adds a deal note, and links the Calendly activity to the deal.

Fluxo de execução

1
Trigger & Parse

Calendly sends a booking webhook. Parse Calendly Event extracts attendee data. Add Deal Owner ID maps the host to the Pipedrive user ID.

2
Person Lookup

Search Pipedrive for existing person by email. If found, check for open deal. If not found, verify phone and create person.

3
Email Verification

Snov.io email verification polling loop (same as Inbound).

4
Deal Create/Update

Create or update deal with owner, stage, and custom fields. Change owner. Add deal note.

5
Activity Linking

Wait for Calendly activity to appear in Pipedrive. Find the activity by type. Link it to the deal.

6
Airtable Sync

Create or update Airtable registration record with deal data.

ATIVO ⚡ Pipedrive deal update trigger (real-time)

Sync CRM with Airtable + Add Time per Stage

Fires on every Pipedrive deal update. Filters to SALES 2026 pipeline only. Syncs deal data to Airtable. Tracks time spent in each funnel stage (Registration, MQL Trying, MQL Contact, SQL, SAL Trial, SAL Commit). Marks deal milestones (MQL, SAL, First Contact, Won, Lost) in Airtable.

Fluxo de execução

1
Trigger & Filter

Pipedrive Trigger fires on any deal change. SALES 2026 only? gate filters out other pipelines.

2
Build & Sync

Build Deal Data assembles the record. Update Sales Database upserts to Airtable.

3
Stage Time Tracking (6 parallel)

For each stage the deal may have left, queries Pipedrive deal history to calculate time spent in that stage. Runs only if the deal was previously in that stage.

4
Milestone Marking

Checks if deal entered MQL, SAL, First Contact, Won, or Lost. Updates Airtable with the milestone timestamp.
